Chopstick is the perfect theme for publishers, bloggers and people who have passion for particular subjects and want to share their thoughts with the world.

Introduction

Chopstick is a magazine theme for Ghost, that offers you flexibility and customizability.
 
This documentation is intended to help you properly install and configure the Chopstick theme on your Ghost site. Kindly go through this file thoroughly, and should you face a challenge beyond the scope of the documentation, do not hesitate to contact me through Github Issues
 
I will do my best to reply as promptly as possible.
 
Also, follow me on Twitter and/ or LinkedIn, to keep up with latest updates and releases.

Installation

Getting Started

If you are new to Ghost, click on the below links to learn more:
  • Ghost Official documentation: http://docs.ghost.org/
  • Ghost Forum: https://ghost.org/forum/

Download

Download the Chopstick project from Github, here https://github.com/kagaim/Chopstick

Configuration

There are a few options, you need to configure before uploading the "chopstick folder" to your website. 
 
Open config.hbs (partials/config.hbs) with your favorite text editor(for example, notepad, notepad++, Sublime text etc).
 
Change the values of the following variables with your own:
  1. Enter your disqus_shortname.
    1. To get started with disqus commenting, click here: https://publishers.disqus.com/engage
    2. Once you sign up, follow the instructions on how to add disqus to your website and you will be given a "shortname"
    3. Copy the "shortname" and paste it here var disqus_shortname = 'PasteDisqusShortnameHere'; in the config.hbs file
  2. Set the Editor's Pick Widget
    1. From this variable (var editors_tag = 'EnterTagName';), in the config.hbs file, enter the name of the tag from which posts will be picked for this widget.
    2. Also, set the number of posts to be displayed using this variable: var editors_tag_post_count = PostCountHere;
  3. Facebook Widget Configuration
    Enter the URL to your Facebook page here: var fb_page_url = 'FacebookPageURL';
  4. Mailchimp Settings
    1. Login or sign up to Mailchimp
    2. Click on lists
    3. If you would like to create a form for an existing list:
      1. Click the drop-down arrow on the right end of the list name
      2. Then click on the "Sign-up forms"
      3. "Select" General Forms
      4. Copy the link in the "Sign-up form URL" field and then paste in a new tab on your browser. It will redirect to the form itself. Then copy the link from address bar and use this link.
    4. If you have created a new list, do the following:
      1. Click "Sign-up Forms" link
      2. Do steps 3 and 4 from above
    5. Once you have copied the link from the address bar, paste it here var mcform_url = 'PasteMailchimpSignupFormLinkHere';
    6. Finally you can change the success message on newsletter sign-up here: var msg_success = 'Please check your inbox to activate your subscription. Thanks!'; or leave it as default.
  5. Social Media Settings
    1. Enter the links to your social media pages as per below:
      1. YouTube: var cp_youtube = 'PasteYourYouTubePageAddressHere';
      2. Instagram: var cp_instagram = 'PasteYourInstagramProfileAddressHere';
      3. Twitter: var cp_twitter = 'PasteYourTwitterPageAddressHere';
      4. Google Plus: var cp_googleplus = 'PasteYourGooglePlusPageAddressHere';
      5. Facebook: var cp_facebook = 'PasteYourFacebookPageAddressHere';
         
        If you do not want to display a particular social media link icon, just leave the variable empty. For example, if I do not want the YouTube icon to display, I will leave the YouTube variable empty like shown below:
         
        var cp_youtube = '';
Once you are through with the settings, in the config.hbs file, save it.
 

Installation 

After making the above changes, do the following:

  1. Move the chopstick folder to your ghost server  "Themes" folder
  2. Restart "Ghost"
  3. Login to "Ghost Admin" and go to "General Settings"
  4. Under the Theme section, select Chopstick from the drop-down list
  5. Click Save on the top right of the page
  6. Visit your website and the theme should be activated

Demo Data

From the Github project page we have added a demo folder.

Inside the folder, you will find chopstick_demo.json file
Setup the demo data using the following instructions:
  1. Go to the labs section of your Ghost installation's settings (http://example.com/ghost/settings/labs/)
  2. Click 'Choose File' under Import
  3. In the resulting file picker, navigate to wherever you extracted the demo content and select the json file.
  4. Click the Import button.

 

Widgets

Chopstick comes with 3 widget areas:
  1. Sidebar
  2. Newsletter
  3. Header Advert

Sidebar Widgets

The sidebar widget area is controlled by the sidebar.hbs file (partials/sidebar.hbs)
This file controls the following widgets:
  1. Editor's Pick Widget: {{> widget-editorspick}}
  2. Advertisement: {{> widget-sidebarad}}
  3. Podcasts: {{> widget-podcasts}}
  4. Facebook Page: {{> widget-facebook}}
  5. Contacts: {{> widget-contacts}}

Order of Widgets Display

If you want to change the order of display of the widgets, you can cut and paste the respective codes. For example, if I want the Contacts to be the first widget shown in the sidebar, I will do the following:
  1. Open sidebar.hbs in my favorite text editor.
  2. Cut "{{> widget-contacts}}" and paste it before "{{> widget-editorspick}}"
  3. Save "sidebar.hbs"
  4. Reload my ghost blog to check the updates
 
Disable a widget
You can also disable a widget by commenting its code. 
For example, if you want to hide {{> widget-facebook}} then add ! like this {{!> widget-facebook}}
Save "sidebar.hbs"
 
Update a Widget
You can update widgets to show your preferred information. Check below for instructions:
 
Editor's Pick Widget
Check configuration instructions, above, on how to update this widget
 
Advertisement Widget
  1. Upload you image advertisement (300*300px) to "assets/images" folder
  2. Go to the "Partials" folder and open "widget-sidebarad.hbs" file in your favorite text editor
  3. On line 5, change the name "advert_300x300.jpg" to the name of your advert "example_ad.jpg"
  4. Save "widget-sidebarad.hbs"
Facebook Widget
Check configuration instructions, above, on how to update this widget
 
Podcasts Widget
  1. Go to the "Partials" folder and open "widget-podcasts.hbs" file in your favorite text editor
  2. Between line 5 and line 9, remove the current "Soundcloud embed code" and paste your "Soundcloud embed code"
  3. Save "widget-podcasts.hbs"
Contacts Widget
  1. Go to the "Partials" folder and open "widget-contacts.hbs" file in your favorite text editor
  2. Change the content in this file to your correct information
  3. Save "widget-contacts.hbs"

Newsletter Widget

How to disable the newsletter widget:
  1. Go to "partials" folder and open "footer.hbs" in your favorite text editor
  2. Between line 1 and line 5 you will see the code that displays the Newsletter Widget: {{> widget-newsletter}}
  3. Disable the widget by commenting it out like so, add ! to {{> widget-newsletter}} so that it looks like this {{!> widget-newsletter}}
  4. Save "footer.hbs"
How to configure the Newsletter Widget
Check configuration instructions, above, on how to update this widget

Header Advert Widget

  1. How to disable the header advert widget:
  2. Go to "partials" folder and open "headerad.hbs" in your favorite text editor
  3. Between line 1 and line 5 you will see the code that displays the Header Advert Widget: {{> widget-headerad}}
  4. Disable the widget by commenting it out like so, add ! to {{> widget-headerad}} so that it looks like this {{!> widget-headerad}}
  5. Save "headerad.hbs"
How to update the Header Advertisement
  1. Upload you image advertisement (728*90px) to "assets/images" folder
  2. Go to the "Partials" folder and open "widget-headerad.hbs" file in your favorite text editor
  3. On line 3, change the name "ad_banner_728x90.jpg" to the name of your advert "example_header_ad.jpg"
  4. Save "widget-headerad.hbs"
